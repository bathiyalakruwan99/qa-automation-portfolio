# Screenshots for Route Optimizer

This folder contains screenshots of the Route Optimizer application and its features.

## How to Add Screenshots

1. **Take a screenshot** of the relevant part of the application or feature.
2. **Save the image** in this `screenshots/` folder.
3. **Update `README.md`** in the parent directory (`route-optimizer/README.md`) to reference your new screenshot.
   - Use Markdown image syntax: `![Alt text](screenshots/your-image-name.png)`
   - Add a brief description below the image.

## Guidelines

- **File Format:** PNG or JPG recommended.
- **Resolution:** Aim for clear, readable images (e.g., 1200-1600px width).
- **Context:** Ensure screenshots clearly illustrate the feature or functionality they represent.
- **Filenames:** Use descriptive, lowercase, hyphen-separated filenames (e.g., `main-interface.png`, `route-alternatives.png`).
