# Demo Videos for Route Optimizer

This folder contains demo videos showcasing the functionality of the Route Optimizer application.

## How to Add Demo Videos

1. **Record your screen** demonstrating the tool's features.
   - Use tools like Windows Game Bar (Win+G), OBS Studio, or similar.
2. **Export the video** in MP4 format.
3. **Save the video file** in this `videos/` folder.
4. **Create a thumbnail image** for your video (optional but recommended) and save it in the `screenshots/` folder.
   - **Naming Convention:** `video-thumbnail-your-video-name.png`
   - **Example:** `screenshots/video-thumbnail-quick-start.png`
5. **Update `README.md`** in the parent directory (`route-optimizer/README.md`) to link to your video.
   - Use Markdown video link syntax: `[![Video Thumbnail](screenshots/your-thumbnail.png)](videos/your-video.mp4)`
   - Add a brief description below the video link.

## Guidelines

- **File Format:** MP4 recommended.
- **Duration:** Keep videos concise (2-5 minutes per demo).
- **File Size:** Aim for reasonable file sizes (e.g., under 50MB per video for GitHub). For larger videos, consider hosting on YouTube/Vimeo and linking.
- **Clarity:** Ensure audio (if any) and visuals are clear.
- **Filenames:** Use descriptive filenames with spaces or hyphens.

## Recommended Video Topics

Consider creating videos for:
