## üìù How to Add Demo Videos

### Option 1: Using Screen Recording Software

1. **Record your screen** using:
   - **Windows:** Xbox Game Bar (Win + G) or OBS Studio
   - **Mac:** QuickTime Player or ScreenFlow
   - **Linux:** OBS Studio or SimpleScreenRecorder

2. **Record each tool demo:**
   - Open the tool in your browser
   - Show the main features and workflow
   - Keep videos concise (2-5 minutes)
   - Show clear navigation and actions

3. **Export videos:**
   - Format: MP4 (recommended) or WebM
   - Resolution: 1280x720 (720p) or 1920x1080 (1080p)
   - Keep file size reasonable (< 50MB per video)

4. **Save videos** with the exact names listed above:
   ```
   videos/dashboard-demo.mp4
   videos/gps-simulator-demo.mp4
   videos/gps-path-builder-demo.mp4
   videos/gps-vehicle-simulator-demo.mp4
   ```

### Option 2: Upload to YouTube/Vimeo

If videos are too large for GitHub, you can:
1. Upload to YouTube or Vimeo (unlisted/public)
2. Update the README.md with YouTube/Vimeo embed links instead
3. Keep thumbnails in the screenshots folder

### Video Content Guidelines

**For Dashboard Demo:**
- Show dashboard overview
- Navigate to each tool
- Explain tool purposes briefly
- Show tool cards and navigation

**For GPS Simulator Demo:**
- Show device data loading
- Configure environment and token
- Start simulation
- Show real-time map updates
- Show statistics panel

**For GPS Path Builder Demo:**
- Show API key setup
- Click waypoints on map
- Show route generation
- Export GPS data
- Explain output format

**For GPS Vehicle Simulator Demo:**
- Show token authentication
- Click on map to move vehicle
- Show GPS data transmission
- Show debug console
- Explain API integration

## üìä Video Specifications

- **Format:** MP4 (H.264 codec) or WebM
- **Resolution:** 1280x720 (720p) minimum, 1920x1080 (1080p) preferred
- **Frame Rate:** 30 FPS
- **Aspect Ratio:** 16:9
- **Duration:** 2-5 minutes per video
- **File Size:** Keep under 50MB per video (or use external hosting)

## üîó Linking Videos in README

Videos are automatically linked in the main README.md once you add:
1. The video file to this folder
2. The corresponding thumbnail to `screenshots/` folder

The README uses markdown image links that will display as clickable video thumbnails.

## üí° Tips for Good Demo Videos

- **Start with a brief intro** (5-10 seconds)
- **Show clear mouse cursor** movement
- **Speak clearly** if adding narration (optional)
- **Highlight important features** with cursor movement
- **End with a summary** or next steps
- **Keep it focused** - one tool per video
- **Test video playback** after uploading
