## üìù How to Add Screenshots

1. **Take screenshots** of each tool interface:
   - Use Windows Snipping Tool (Win + Shift + S)
   - Or any screenshot tool
   - Capture the full application window

2. **Save files** with the exact names listed above:
   ```
   screenshots/dashboard-main.png
   screenshots/gps-simulator-main.png
   screenshots/gps-path-builder-main.png
   screenshots/gps-vehicle-simulator-main.png
   screenshots/multi-device-combiner-main.png
   ```

3. **File format:** PNG or JPG
4. **Recommended size:** 1200-1600px width
5. **Quality:** Clear and readable
