# Adding Screenshots

## How to Add a Screenshot

1. **Take a screenshot** of the Excel File Corrector GUI
   - Press `Windows + Shift + S` (Windows) or `PrtScn` key
   - Or use any screenshot tool

2. **Save the screenshot** in this folder (`screenshots/`) with the name:
   - `main-gui.png` - Main desktop GUI interface
   - `options-dialog.png` - Processing options dialog
   - `web-interface.png` - Web interface (if applicable)

3. **Recommended format:**
   - File format: PNG (best quality) or JPG
   - File name: descriptive (e.g., `main-gui.png`)
   - Size: Try to keep under 2MB for fast loading

4. **The README.md will automatically display it** once you save the image file
