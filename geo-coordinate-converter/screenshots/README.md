# Screenshots

This folder contains screenshots of the Location Data Processor application.

## Screenshot Files

### Main GUI Screenshot
- **`gro cordiante ui.png`** - Main desktop GUI interface showing:
  - Application window with title "Location Data Processor"
  - Location File input field with Browse button
  - Order File input field with Browse button
  - Output JSON File input field with Save As button
  - Process Files button
  - Processing Log text area

### Output Files Screenshots

**JSON Output:**
- **`geo crodinate jason.png`** - Example of generated JSON file showing:
  - Location data in JSON array format
  - Each location with name, locationReferenceId, latitude, and longitude

**Excel Output:**
- **`geo cordinate excel.png`** - Example of generated Excel file showing:
  - Location Name column
  - Location Reference ID column
  - Latitude column
  - Longitude column
  - Google Maps Format column (ready to search in Google Maps!)
