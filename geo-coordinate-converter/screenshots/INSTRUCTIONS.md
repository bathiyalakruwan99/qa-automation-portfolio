# Adding Screenshots to Location Data Processor

## Where to Save Screenshots

Save your screenshots in this folder: **`screenshots/`**

## Main Screenshot

**Required:**
- **Filename:** `main-gui.png`
- **Description:** Main desktop GUI interface showing:
  - Application window title "Location Data Processor"
  - Location File input field with Browse button
  - Order File input field with Browse button
  - Output JSON File input field with Save As button
  - Process Files button
  - Processing Log text area

## How to Add Screenshots

1. **Take a screenshot** of the application GUI
   - Use Windows Snipping Tool or any screenshot tool
   - Capture the full application window

2. **Save the file** to this folder:
   ```
   geo-coordinate-converter/screenshots/main-gui.png
   ```

3. **Verify** it appears in the README.md
   - The README.md already has the image reference
   - It will automatically display once you add the file

## Screenshot Guidelines

- **Format:** PNG or JPG
- **Recommended size:** 800-1200px width
- **Quality:** Clear and readable
- **Content:** Show the main GUI interface with all controls visible

## Optional Additional Screenshots

You can also add:
- `processing-log.png` - Example of processing output
- `output-example.png` - Example of generated JSON/Excel files

These will appear in the README if you add image references.
